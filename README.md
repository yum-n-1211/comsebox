### テーマを選んだ理由
<女性にとっての化粧とは、「オンとオフの切り替え」「好きな自分になる」「気持ちが明るくなる」など大きな意義があります。
自分のお気に入りコスメを紹介することでコスメ好きの方の新たなコミュニケーションのきっかけとなったり、
自分のご褒美に新しいコスメを買う時に他者の評価を見ることができるサイトです。
家事や仕事に毎日を頑張る人の息抜き・気分転換の場所を提供できればと思い、このテーマを選定しました。>
